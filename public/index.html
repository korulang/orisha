Orisha

A compile-time web framework for Koru.

Zero runtime overhead. Routes, content, etags - all computed at compile time and embedded in the binary.

How It Works

Declare routes with [norun] events:

~route(GET /) {
    "file": "public/index.html"
}

At compile time, Orisha:

  • Reads the file
  • Computes SHA256 etag
  • Generates optimized routing code
  • Embeds everything in the binary
