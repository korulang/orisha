# Hello World Example

A minimal Eshu + Oya application demonstrating basic HTTP routing and templating.

## Structure

```
hello/
├── main.kz         # Entry point
├── routes.kz       # Route definitions
└── README.md       # This file
```

## Running

```bash
cd examples/hello
koruc main.kz
./main
```

Visit `http://localhost:3000`

## What This Demonstrates

- Basic HTTP server setup
- Simple route definition
- Template rendering with Oya
- Type-safe request/response handling
